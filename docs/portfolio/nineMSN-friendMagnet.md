2008

FriendMagnet was a flash based game to support over 300k+ simultaneous NineMSN Messenger users.

The project consisted of an ASP.NET based backend utilizing LINQ which was only new at the time. Helped architect solution and develop.

- ASP.NET 3
- LINQ 2 SQL
- Google Contacts API's
- Messenger API's
- Various code tasks
